


# amduspage

a simple messenger pagebot.

## setup
1. clone the repository
2. run `npm install`
3. configure environment variables
4. start with `node index.js`

## environment variables
- page_access_token
- verify_token
- mongodb_uri
- admins

## features
- zero-prefix logic
- ai commands
- media tools
- utilities
- admin controls

## commands
- ai: amdus, gemini, copilot
- media: alldl, dalle, pinterest
- utility: google, wiki, translate
- fun: joke, bible, 48laws
